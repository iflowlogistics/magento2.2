Modulo Iflow

Encontraran dos carpetas, una con el nombre "Con andreani" y otra "Sin andreani".
La diferencia es que la carpeta "Con andreani" tiene el modulo adaptado a los
atributos de direccion que se suelen utilizar como altura, piso, departamento y
observaciones. Optamos por nombrarla de esta forma ya que este modulo es conocido
por agregar dichos atributos de direccion. La realidad es que esa solucion
aplica a todo sitio que tenga dichos atributos.

Por otro lado la carpeta "Sin andreani" tiene la solucion para aquellos sitios
que se manejan con los atributos nativos de magento.
